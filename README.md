




### How to use
1. Select export node (all child objects of this node are exported)
2. Set its mode to Export Node using "LR Exporter" panel or in shift + e menu
3. Export - select any object in hierarchy. Exporter will crawl up in object hierarchy until it reaches object which is marked as Export Node. This object and its children are exported with settings in exporter panel.

### Other info
- Addon has a custom menu under shift + e hotkey

### Supported
- Basic export features for Unreal like reset to world 0 and reset rotation
- Creating subdirectories for each export node
- Custom python scripts during export process for custom workflows
